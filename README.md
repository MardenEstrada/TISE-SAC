# TISE SAC — Landing page

Sitio corporativo estático de una sola página. Sin backend, sin API, listo para deploy.

## Secciones

- `#inicio` — Hero
- `#servicios` — Servicios
- `#nosotros` — Empresa, historia, misión
- `#proyectos` — Tecnologías y casos
- `#contacto` — Datos de contacto

## Modo claro / oscuro

Toggle en el navbar. La preferencia se guarda en `localStorage` (`tise-theme`).

## Comandos

```bash
npm install
npm run dev
npm run build
```

Sube la carpeta `dist/` a cualquier hosting estático.
